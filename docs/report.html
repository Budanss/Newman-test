Запрос отправляется методом POST на адрес `https://sklad.testerhub.ru/wms/v1/auth/login` и требует предоставления email и пароля пользователя. #### Входные параметры Тело запроса должно содержать следующие параметры в формате JSON: - **email** (строка): Адрес электронной почты пользователя, используемый для входа в систему. - **password** (строка): Пароль пользователя, соответствующий указанному адресу электронной почты. Эта информация может быть использована для дальнейшего взаимодействия с другими конечными пунктами API, которые могут возвращать аналогичные структуры данных.
